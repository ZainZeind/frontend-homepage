Modern Homepage — TailwindCSS + Vite

Modern Homepage adalah landing page sederhana dengan desain modern, efek glassmorphism, dan UI minimalis. Dibangun menggunakan Vite, JavaScript, dan TailwindCSS.

Cara Menjalankan Project
1. Clone Repository
git clone https://github.com/ZainZeind/frontend-homepage.git

2. Masuk ke folder project
cd frontend-homepage

3. Install dependency
npm install

4. Jalankan server development
npm run dev

5. Akses di browser
http://localhost:5173

Keterangan

Project ini dibuat sebagai tugas frontend menggunakan TailwindCSS, Vite, dan JavaScript.
Desain menggunakan gaya glassmorphism, ikon transparan, dan layout modern agar terlihat profesional.

Struktur utama project:

src/ → Source code utama

public/ → Asset statis

tailwind.config.js → Konfigurasi Tailwind

vite.config.js → Konfigurasi Vite

package.json → Dependency project
